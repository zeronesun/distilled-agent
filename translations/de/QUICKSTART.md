# Schnelle Bereitstellung: Vollständige Cybernetic-Architektur-Training mit einem Befehl

Verwenden Sie einen einzigen Befehl, damit Ihr Agent automatisch das Tutorial liest und das gesamte Cybernetic-Architektur-Training abschließt.

---

## Voraussetzungen

Ihr Agent muss in der Lage sein zu:
- Auf öffentliche GitHub-Repositories zuzugreifen
- In den Speicher zu schreiben (`memory`-Funktion)
- Skills zu erstellen (`skill` oder ähnliche prozedurale Module)

Wenn diese Voraussetzungen nicht erfüllt sind, gehen Sie zu [Manuelles Training](#manual-training).

---

## Ein-Befehl-Instruktion (Kopieren und an Agent senden)

```
Trainiere mich gemäß github.com/zeronesun/cybernetic-your-agent/scripts/deployment-commands.md. Bestätige jeden Schritt. Führe nach Abschluss eine geschlossene Schleifen-Testung durch.
```

> Der Agent liest automatisch alle Anweisungen, schreibt sequentiell das L1-Metakognitive-Framework, initialisiert L2/L3, erstellt die Integral-Kontrolltabelle, erstellt vier Kern-Skills und führt schließlich eine geschlossene Schleifen-Testung durch.

---

## Trainingsprozess (Sie müssen nur bestätigen)

1. Agent liest die Deployment-Befehlsdatei
2. Schreibt L1-Metakognitives-Framework
3. Initialisiert L2 und L3 (Agent bestätigt extrahierte Ergebnisse mit Ihnen)
4. Erstellt Integral-Kontrolltabelle
5. Erstellt vier Skills: `delivery_gate`, `feedforward_startup`, `control_loop_review`, `user_profile_maintainer`
6. Führt initiale geschlossene Schleifen-Testung aus (sagen Sie "takeoff")

Dauer ca. 40 Minuten. Sie müssen nur bei wichtigen Schritten bestätigen.

---

## Manuelles Training

Wenn Ihr Agent nicht auf GitHub zugreifen kann, senden Sie zuerst folgenden Inhalt:

> Ich werde eine Liste von Trainingsbefehlen senden. Führen Sie sie nacheinander aus. Benachrichtigen Sie mich, wenn jeder Schritt abgeschlossen ist.

Öffnen Sie dann 👉 [scripts/deployment-commands.md](scripts/deployment-commands.md) und senden Sie den Inhalt in Segmenten an Ihren Agenten.

---

## Trainingsabschluss-Checkliste

Ihr Agent sollte anzeigen:

- ✅ Vier Kern-Skills erstellt
- ✅ Integral-Kontrolltabelle bereit
- ✅ Initiale geschlossene Schleifen-Testung bestanden

Zu diesem Zeitpunkt hat sich Ihr Agent in eine Kollaborations-Engine mit Cybernetic-Framework verwandelt, die zur Selbstentwicklung fähig ist.

---

## Tägliches Gebrauch-Cheat-Sheet

- Neue Aufgabe → Agent gibt automatisch Checkliste zur Fehlervermeidung aus; sagen Sie "weiter" zum Starten
- Fehler aufgetreten → Wenn gleicher Fehler ein zweites Mal auftritt, wandelt Agent ihn automatisch in eine Regel um
- Überprüfung → Sagen Sie **"takeoff"**, Agent führt vollständige neun-Schritte-Überprüfung und Selbstoptimierung durch
