# Quick Deployment: Complete Cybernetic Architecture Training with One Command

Use a single command to make your Agent automatically read the tutorial and complete the entire cybernetic architecture training.

---

## Prerequisites

Your Agent must be able to:
- Access GitHub public repositories
- Write to memory (`memory` capability)
- Create skills (`skill` or similar procedural modules)

If these requirements are not met, skip to [Manual Training](#manual-training).

---

## One-Command Instruction (Copy and Send to Agent)

```
Train me according to github.com/zeronesun/cybernetic-your-agent/scripts/deployment-commands.md. Confirm each step. Perform closed-loop testing upon completion.
```

> The Agent will automatically read all instructions, sequentially write the L1 metacognitive framework, initialize L2/L3, establish the integral control table, create four core skills, and finally execute a closed-loop test.

---

## Training Process (You Only Need to Confirm)

1. Agent reads the deployment command file
2. Writes L1 metacognitive framework
3. Initializes L2 and L3 (Agent confirms提炼 results with you)
4. Establishes integral control table
5. Creates four skills: `delivery_gate`, `feedforward_startup`, `control_loop_review`, `user_profile_maintainer`
6. Runs initial closed-loop testing (say "takeoff")

Takes approximately 40 minutes. You only need to confirm at key steps.

## Manual Training

If your Agent cannot access GitHub, first send it the following content:

> I will send a list of training commands. Execute them one by one. Notify me when each step is completed.

Then open 👉 [scripts/deployment-commands.md](scripts/deployment-commands.md) and send the content to your Agent in segments.
