# 빠른 배치: 한 문장으로 에이전트가 제어론 아키텍처 훈련 완료

한 문장 명령으로 에이전트가 튜토리얼을 자동으로 읽고 완전한 제어론 아키텍처 훈련 완료.

---

## 적용 조건

에이전트가 다음을 수행할 수 있어야 함:
- GitHub 공개 저장소 접근
- 기억 저장 (`memory` 기능)
- 스킬 생성 (`skill` 또는 유사 절차적 모듈)

조건 미충족 시 [수동 훈련](#수동-훈련)으로 이동.

---

## 한 문장 명령 (에이전트에 복사하여 전송)

```
github.com/zeronesun/cybernetic-your-agent 의 scripts/deployment-commands.md 에 따라 나를 훈련시켜. 각 단계 확인 완료 후 폐쇄 루프 테스트.
```

> 에이전트가 모든 명령을 자동으로 읽고, 순서대로 L1 메타인지 프레임워크에 작성하고, L2/L3를 초기화하고, 적분 제어 테이블을 구축하고, 네 가지 핵심 스킬을 생성한 뒤, 최종적으로 한 번의 폐쇄 루프 테스트를 실행함.

---

## 훈련 과정 (확인만 하면 됩니다)

1. 에이전트가 배치 명령 파일 읽음
2. L1 메타인지 프레임워크 작성
3. L2와 L3 초기화 (에이전트가 추출 결과를 확인에 알림)
4. 적분 제어 테이블 구축
5. 네 가지 스킬 생성: `delivery_gate`, `feedforward_startup`, `control_loop_review`, `user_profile_maintainer`
6. 첫 번째 폐쇄 루프 테스트 실행 ("이륙" 말하기)

전체 약 40분, 중요 단계 확인만 필요.

---

## 수동 훈련

에이전트가 GitHub에 액세스할 수 없는 경우, 먼저 아래 내용을 에이전트에 복사:

> 훈련 명령 목록을 발송하니 각 단계를 순차적으로 실행. 단계 완료마다 알림.

그다음 👉 [scripts/deployment-commands.md](scripts/deployment-commands.md) 를 열고 내용을 구분하여 에이전트에 복사.

---

## 훈련 완료 표지

에이전트가 표시해야 함:

- ✅ 네 가지 핵심 스킬 생성 완료
- ✅ 적분 제어 테이블 준비 완료
- ✅ 첫 번째 폐쇄 루프 테스트 통과

이 시점에 에이전트는 제어론을 골격으로 하고 자기 진화 가능한 협업 엔진으로 변모해야 함.

---

## 일상 사용 속노트

- 새로운 과제 → 에이전트가 자동으로 문제 회피 목록 출력, "바로 하면" 시작
- 오류 발생 → 동류 문제 2번째 발생 시, 에이전트가 자동으로 규칙으로 내려감
- 복기 → **"이륙"** 말하면, 에이전트가 전체 9스텝 복기 실행 후 자기 최적화
