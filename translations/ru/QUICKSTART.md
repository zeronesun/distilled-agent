# Быстрое развертывание: Тренировка кибернетической архитектуры одной фразой

Одной командой заставьте Агента автоматически прочитать учебник и завершить полное обучение кибернетической архитектуре.

---

## Условия применимости

Ваш Агент должен быть способен к:
- Доступ к публичным репозиториям GitHub
- Запись в память (функция `memory`)
- Создание навыков (`skill` или аналогичных процедурных модулей)

Если это не так, перейдите к [Ручное обучение](#ручное-обучение).

---

## Команда в Одну Фразу (Скопировать и Отправить Агенту)

```
Обучи меня согласно scripts/deployment-commands.md из github.com/zeronesun/cybernetic-your-agent. Подтверждай каждый шаг, а затем выполни тест с обратной связью в конце.
```

> Агент автоматически прочитает все инструкции, последовательно напишет метакогнитивную структуру L1, инициализирует L2/L3, установит интегральную таблицу управления, создаст четыре базовых навыка, и в завершение выполнит единичный тест с обратной связью.

---

## Процесс Обучения (Вам Нужно Только Подтверждать)

1. Агент читает файл инструкций по развертыванию
2. Пишет метакогнитивную структуру L1
3. Инициализирует L2 и L3 (Агент подтверждает вам результаты извлечения)
4. Устанавливает интегральную таблицу управления
5. Создает четыре навыка: `delivery_gate`, `feedforward_startup`, `control_loop_review`, `user_profile_maintainer`
6. Выполняет первый тест с обратной связью (скажите "взлёт")

Полный процесс занимает около 40 минут, вам нужно только подтверждать на ключевых этапах.

---

## Ручное обучение

Если Агент не может получить доступ к GitHub, сначала отправьте ему следующее:

> Я отправлю список инструкций по обучению, пожалуйста, выполняйте их одну за другой. Сообщайте мне о каждом завершённом этапе.

Затем откройте 👉 [scripts/deployment-commands.md](scripts/deployment-commands.md), скопируйте содержимое по сегментам и отправьте Агенту.

---

## Индикаторы Завершённого Обучения

Агент должен отобразить:

- ✅ Четыре базовых навыка созданы
- ✅ Интегральная таблица управления готова
- ✅ Первый тест с обратной связью успешно выполнен

На этом этапе ваш Агент превратился в кооперативный движок, способный к автономной эволюции, с кибернетикой в качестве каркаса.

---

## Запомните Для Ежедневного Использования

- Новая задача → Агент автоматически отображает список избегания ловушек, скажите "делай напрямую" для начала
- Ошибка → Когда аналогичная проблема возникает во второй раз, Агент автоматически превращает её в правило
- После-действенная ревизия → Скажите **"взлёт"**, Агент выполняет полную ревизию в девять этапов и самооптимизируется
